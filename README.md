# GreenLens AI

GreenLens is an AI-powered smart waste management platform that helps detect, report, and manage garbage in cities. The system connects citizens, drivers, and municipalities to ensure faster waste collection and cleaner environments.

## 🚀 Features

- AI-based waste detection
- Citizens can report garbage locations
- Driver dashboard for waste collection
- Municipality admin dashboard
- Real-time tracking of waste reports
- Smart city waste management solution

## 👥 Users

1. **Citizens**
   - Upload waste images
   - Report garbage locations

2. **Drivers**
   - Receive collection tasks
   - Track waste pickup locations

3. **Municipality Admin**
   - Manage waste reports
   - Monitor drivers
   - Track city cleanliness

## 🛠 Tech Stack
- AI Model: Waste detection using AI
- Platform: Web Application

## 📌 Problem Statement

Many cities struggle with unmanaged waste due to slow reporting and inefficient waste collection systems.

## 💡 Our Solution

GreenLens provides an AI-powered reporting and management system that connects citizens and municipalities for faster and smarter waste management.

## 🌍 Impact

- Cleaner cities
- Faster waste reporting
- Improved municipal management
- Better citizen participation
